# Flask Authentication App (Lab 3)

##  Описание проекта

Данное веб-приложение разработано в рамках лабораторной работы №3 и реализует механизм аутентификации пользователей с использованием фреймворка Flask и библиотеки Flask-Login.

---

##  Функционал

### 1. Счётчик посещений

* Реализован с использованием объекта `session`
* Отображает количество посещений страницы пользователем
* Данные хранятся индивидуально для каждого пользователя

---

### 2. Аутентификация пользователей

* Реализована с использованием Flask-Login
* Форма входа с полями:

  * логин
  * пароль
  * чекбокс "Запомнить меня"
* Тестовые данные:

  * **логин:** user
  * **пароль:** qwerty
* После входа:

  * отображается сообщение об успешной авторизации
  * происходит редирект на главную страницу
* При ошибке:

  * отображается сообщение о неверных данных

---

### 3. Секретная страница

* Доступна только авторизованным пользователям
* Защищена с помощью `@login_required`
* Неавторизованные пользователи перенаправляются на страницу входа
* После входа происходит возврат на запрашиваемую страницу

---

### 4. Навигация

* Navbar отображает разные ссылки в зависимости от статуса пользователя:

  * Авторизованный:

    * Главная
    * Секретная
    * Выйти
  * Неавторизованный:

    * Главная
    * Войти

---

## Установка и запуск

### 1. Клонировать репозиторий

```bash
git clone https://github.com/your-username/flask-auth-lab3.git
cd flask-auth-lab3
```

### 2. Создать виртуальное окружение

```bash
python3 -m venv venv
source venv/bin/activate
```

### 3. Установить зависимости

```bash
pip install -r requirements.txt
```

### 4. Запустить приложение

```bash
python app.py
```

### 5. Открыть в браузере

```
http://127.0.0.1:5000
```
